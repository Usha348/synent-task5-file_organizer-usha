# File Organizer (Python)

## Description
This project is a Python program that automatically organizes files in a folder based on their file type (extension).

## Features
- Reads all files from a folder
- Identifies file extensions (jpg, pdf, txt, etc.)
- Creates folders automatically
- Moves files into respective folders

## Technologies Used
- Python
- os module
- shutil module

## How to Run
1. Open terminal in project folder
2. Run command:
   python file_organizer.py
3. Enter the folder path when asked

## Example
Before:
photo.jpg, file.pdf

After:
jpg/photo.jpg  
pdf/file.pdf
